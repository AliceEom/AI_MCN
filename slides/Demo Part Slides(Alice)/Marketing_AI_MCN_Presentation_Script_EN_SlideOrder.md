## Live App Tab 3: Network Studio

“This tab explains creator ecosystem structure.”

“Control meanings:
Nodes in View controls graph size.
Min Edge Strength removes weak relationships.
Communities to Show controls cluster summary detail.
Include micro/isolated adds tiny clusters.”

“Graph meaning:
One node = one creator channel.
Node size = final score magnitude.
Node color = community ID.
Edge = topical connection from shared tags.”

“I can click a node from the selector and inspect:
selected channel,
community,
final score,
and median views.”

“Community Diagnostics shows distribution.
Graph Meta shows nodes, edges, dropped over-common tags.
Bias report shows overlap with popularity-only ranking.
So we can check diversity and bias risk.”

“Track 1 is Scale and Awareness.
These creators expand discovery.”

“Track 2 is Education and Support.

“Track 3 is Conversion and Fit.
