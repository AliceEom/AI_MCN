AI-MCN: AI-Augmented Influencer Matching for Beauty Campaigns

MSIS 521 Course Project | Team Presentation (15 minutes)

Case: Beauty of Joseon (BOJ), U.S. sunscreen campaign prototype

What we built: End-to-end Streamlit app + Python AI pipeline for influencer recommendation, evaluation, and decision support.

SNA + Network Centrality TF-IDF + Semantic Matching Linear/LASSO/Ridge/CART/RF/LightGBM 5-Fold GroupCV + SHAP ROI Simulator + Memo Export

Problem and Scope

Business Problem

  • Influencer discovery is often manual, slow, and popularity-biased.
  • Teams need transparent, evidence-based creator selection.
  • Campaign decisions should connect to measurable outcomes.

Scope Discipline

  • In-scope: full matching prototype + demo + evaluation.
  • Out-of-scope: production MLOps and full live social ingestion.
  • Designed to be credible for a quarter-long class project.

Method Pipeline

  2. Channel aggregation and text profile generation
  3. Graph construction + SNA scoring + community detection
  4. TF-IDF relevance + semantic/tone enrichment for top candidates
  5. ML benchmark (6 models, GroupKFold 5-fold)
  6. Evidence penalty + diversity-aware ranking
  7. ROI simulation + strategy generation + executive memo export

ML Evaluation Results

Best Model: LightGBM

  • RMSE: 0.00996
  • Baseline RMSE: 0.03800
  • Improvement vs baseline: 73.8%
  • Group-based CV prevents channel leakage across folds.

Explainability (SHAP)

Why this matters

  • Model behavior is auditable and easier to explain in stakeholder meetings.
  • Supports confidence in ranking decisions beyond black-box outputs.
  • Useful for diagnosing overfitting or unstable features.

Guardrails Against Bad Recommendations

What we enforce

  • Low-evidence channels are down-weighted even if keyword match is high.
  • Diversity selection limits same-cluster over-concentration.
  • Bias report compares degree-only vs hybrid ranking.

Bias metric: overlap between degree-only top-10 and hybrid top-10 = 2/10.

Network Diversity Diagnostics

Limitations and Future Work

Current limits

  • Prototype relies on pre-collected YouTube data.
  • ROI is scenario-based, not causal inference.
  • Some cluster concentration remains in skincare-heavy niches.

Next steps

  • Expand to TikTok/Instagram data connectors.
  • Add online experiment loop for closed-loop performance learning.
