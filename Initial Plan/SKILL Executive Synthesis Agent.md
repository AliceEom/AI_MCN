name: SKILL_ExecutiveSynthesisAgent
description: >
  Synthesises all AI-MCN pipeline outputs into a single executive-grade Brand
  Partnership Recommendation Memo. Use this subagent at Step 7I of the AI-MCN
  orchestrator, called once sequentially after all 5 Content Strategy Agents have
  completed. It is the final intelligence layer of the pipeline. Receives top5_df,
  strategy_texts dict, roi_result dict, bias_report dict, and brand_params as
  inputs. Produces a structured Markdown memo under 600 words containing an
  executive summary, priority recommendation with data citations, 4-week campaign
  structure, risk flags from red_flags fields, and 3 next steps. Output is
  rendered in Streamlit via st.markdown() and offered as a downloadable file.
  Must always run after all other subagents complete — it has hard dependencies
  on Channel Enrichment Agent outputs (alignment_rationale, red_flags) and
  Content Strategy Agent outputs (strategy_texts). Never run in parallel or
  before Step 6B finishes.

# Executive Synthesis Agent

## Role in Pipeline

```
Step 4B — Brand Intelligence Agent  ──────────────────────────────┐
Step 5F — Channel Enrichment Agents (×20) — red_flags, rationale  │
Step 6A — ROI simulation result                                    ├──► [THIS AGENT]
Step 6B — Content Strategy Agents (×5) — strategy_texts dict      │
Step 5E — Competitive bias report                                  │
Step 7I — [THIS AGENT] → brand_memo.md (Markdown)                 ┘
Step 7I — st.markdown(brand_memo) + st.download_button()
```

**Execution:** Sequential · 1 call · `llama-3.3-70b-versatile` (Groq) · max_tokens=900

**Hard dependency order:** This agent must be the last subagent called. It reads
outputs from all 4 prior pipeline stages. Calling it before Step 6B completes will
produce a memo with missing strategy summaries.

**Why this agent, not a dashboard summary:** By Step 7H the dashboard displays five
separate strategy documents, an ROI simulation, a bias audit, five score breakdowns,
and a network graph. A brand manager or CMO should not be required to mentally
synthesize these into a single recommendation. This agent does that synthesis and
produces the document a real decision-maker would actually share with their team.
It is also the strongest demonstration of LLM value to a non-technical evaluator —
the memo is legible business output, not a chart.

## Output Contract

A single Markdown string under 600 words with exactly these 5 sections:

```markdown
# Brand Partnership Recommendation Memo
**Brand:** {name}  |  **Date:** {date}  |  **Budget:** ${budget:,}

## Executive Summary
[3 sentences max. Why these 5 influencers, what ROA to expect, one key insight.]

## Priority Recommendation
[One-paragraph justification for the top pick. Must cite SNA score, semantic
alignment score, and tone match score by name and value.]

## Campaign Structure
[4-week deployment plan across all 5 influencers. Format as a simple table or
numbered list. Reference specific content concepts from the strategy texts.]

## Risk Flags
[Bullet list of red_flags surfaced by Channel Enrichment Agents. If none, state
"No material risk flags identified." Do not omit this section.]

## Next Steps
1. [Concrete action item 1]
2. [Concrete action item 2]
3. [Concrete action item 3]
```

**Strict output rules:**
- Total word count must be under 600 words
- Every data claim must cite a specific number from the inputs (never say "high engagement" — say "1.8% engagement ratio")
- Priority Recommendation must name and numerically cite all three key scores
- Risk Flags section must be present even if empty
- No marketing filler phrases: "leverage synergies", "unlock potential", "holistic approach", etc.

## Implementation

```python
# orchestrator.py
from groq import Groq
import json, os
import pandas as pd

SYNTHESIS_SYSTEM = """
You are a Chief Marketing Officer writing a concise, executive-grade influencer
partnership recommendation memo. You receive structured data outputs from an AI
matching system and translate them into a clear, actionable business document.
Write in confident, direct prose. Use Markdown headers and tables where appropriate.
Never pad with generic marketing language. Every sentence must reference specific data.
"""

SYNTHESIS_PROMPT = """
You have completed an AI-powered influencer matching analysis for the following brand.
Synthesise all outputs into a one-page Brand Partnership Recommendation Memo.

## Brand
{brief}

## Top 5 Recommended Influencers
{top5_table}

## ROI Simulation (at ${budget:,} budget)
- Estimated Impressions: {impressions:,}
- Estimated Clicks: {clicks:,}
- Estimated Conversions: {conversions:,}
- Expected ROA: {roa_expected}x (range: {roa_low}x - {roa_high}x)

## AI Competitive Differentiation
{bias_narrative}

## Content Strategy Summaries (one sentence each)
{strategy_summaries}

## Memo Requirements
1. **Executive Summary** (3 sentences max) - why these 5 influencers, what ROA to expect
2. **Priority Recommendation** - single top pick with one-paragraph justification
   citing SNA score, semantic alignment score, and tone match score by name and value
3. **Campaign Structure** - how to deploy all 5 across a 4-week campaign timeline
4. **Risk Flags** - any red flags surfaced by the AI analysis
5. **Next Steps** - 3 concrete action items for the brand team

Format as clean Markdown. Keep the entire memo under 600 words.
"""


def _build_top5_table(top5_df: pd.DataFrame) -> str:
    cols = [
        'title', 'subscriber_count', 'confidence_score_final',
        'semantic_score', 'engagement_ratio', 'alignment_rationale'
    ]
    existing = [c for c in cols if c in top5_df.columns]
    return top5_df[existing].to_markdown(index=False)


def _build_strategy_summaries(top5_df: pd.DataFrame, strategy_texts: dict) -> str:
    lines = []
    for i, row in top5_df.iterrows():
        text    = strategy_texts.get(row['channel_id'], '')
        summary = text[:200].strip().split('\n')[0].lstrip('#').strip()
        lines.append(f"- **{row['title']}:** {summary}")
    return "\n".join(lines)


def run_executive_synthesis_agent(
    top5_df:        pd.DataFrame,
    strategy_texts: dict,
    roi_result:     dict,
    bias_report:    dict,
    brand_params:   dict,
) -> str:
    """
    Generate the executive Brand Partnership Recommendation Memo.

    This is the final subagent in the pipeline. Call only after all
    Content Strategy Agents (Step 6B) have completed.

    Returns:
        Markdown string. Saved to cache/brand_memo_{hash}.md.
    """
    cache_path = f"cache/brand_memo_{hash(brand_params['brief_md'])}.md"
    os.makedirs("cache", exist_ok=True)

    if os.path.exists(cache_path):
        with open(cache_path) as f:
            return f.read()

    prompt = SYNTHESIS_PROMPT.format(
        brief             = brand_params["brief_md"],
        top5_table        = _build_top5_table(top5_df),
        budget            = int(brand_params.get("budget", 0)),
        impressions       = roi_result.get("impressions", 0),
        clicks            = roi_result.get("clicks", 0),
        conversions       = roi_result.get("conversions", 0),
        roa_expected      = roi_result.get("roa_expected", "N/A"),
        roa_low           = roi_result.get("roa_low", "N/A"),
        roa_high          = roi_result.get("roa_high", "N/A"),
        bias_narrative    = bias_report.get("narrative", ""),
        strategy_summaries= _build_strategy_summaries(top5_df, strategy_texts),
    )

    try:
        client   = Groq()
        response = client.chat.completions.create(
            model      = "llama-3.3-70b-versatile",
            max_tokens = 900,
            messages   = [
                {"role": "system", "content": SYNTHESIS_SYSTEM},
                {"role": "user",   "content": prompt},
            ],
        )
        memo = response.choices[0].message.content
        with open(cache_path, "w") as f:
            f.write(memo)
        return memo

    except Exception as e:
        print(f"[ExecutiveSynthesisAgent] Failed: {e}")
        return _fallback_memo(top5_df, roi_result, brand_params, e)


def _fallback_memo(
    top5_df:     pd.DataFrame,
    roi_result:  dict,
    brand_params: dict,
    error:       Exception,
) -> str:
    """
    Minimal structured memo built from raw data when API fails.
    Preserves dashboard usability during demo if synthesis call fails.
    """
    top_pick = top5_df.iloc[0] if not top5_df.empty else {}
    return f"""
# Brand Partnership Recommendation Memo
**Brand:** {brand_params.get('name', 'N/A')}  |  **Budget:** ${brand_params.get('budget', 0):,.0f}

## Executive Summary
Analysis complete. Top recommendation: **{top_pick.get('title', 'N/A')}**
(confidence score: {top_pick.get('confidence_score_final', 0):.2f}).
Expected ROA: {roi_result.get('roa_expected', 'N/A')}×
(range: {roi_result.get('roa_low', 'N/A')}× – {roi_result.get('roa_high', 'N/A')}×).

## Priority Recommendation
See score breakdown in dashboard for full analysis.

## Risk Flags
Synthesis API unavailable: `{error}`. Review red_flags column in the score table.

## Next Steps
1. Review the Top 5 score breakdown in the dashboard.
2. Select influencers and download individual strategy briefs.
3. Re-run synthesis when API recovers.
""".strip()
```

## Orchestrator Integration

```python
# orchestrator.py → run_full_pipeline()

# Step 7I — called last, after all other subagents complete
brand_memo = run_executive_synthesis_agent(
    top5_df        = top5_df,
    strategy_texts = strategy_texts,
    roi_result     = roi_result,
    bias_report    = bias_report,
    brand_params   = brand_params,

# app.py — Section 7I rendering
with st.expander("📄 Executive Brand Partnership Memo", expanded=True):
    st.markdown(brand_memo)
    st.download_button(
        label     = "⬇ Download Memo",
        data      = brand_memo,
        file_name = f"{brand_params['name'].replace(' ', '_')}_partnership_memo.md",
        mime      = "text/markdown",
    )
```

## Data Preparation Notes

The two helper functions in the implementation are not optional — the prompt template
cannot be filled without them:

**`_build_top5_table(top5_df)`** — converts the DataFrame to a Markdown table using
`pd.DataFrame.to_markdown()`. Requires `tabulate` to be installed:
```
pip install tabulate
```

**`_build_strategy_summaries(top5_df, strategy_texts)`** — extracts the first
non-empty line from each strategy text as a one-sentence summary. The model uses
these summaries to write the Campaign Structure section without needing the full
5×1500-token strategy texts in its context window.

---

## Common Failure Modes

| Failure | Symptom | Resolution |
|---------|---------|------------|
| Called before Step 6B completes | `strategy_summaries` is empty or partial | Enforce call order in `run_full_pipeline()` |
| `top5_df` missing `confidence_score_final` | Markdown table has wrong columns | Check that Step 5F ran; fall back to `confidence_score` |
| `tabulate` not installed | `to_markdown()` raises ImportError | `pip install tabulate` in requirements.txt |
| Memo exceeds 600 words | Quality issue — model padding | Re-run with a stricter system prompt addition: "Your response must be under 600 words. Stop after Next Steps." |
| Cache stale after brand change | Old memo served for new brand | Cache key includes brief hash — changing brief_md invalidates cache automatically |
| `_build_top5_table` crashes on empty `top5_df` | Pipeline error before this agent ran | Guard: `if top5_df.empty: return _fallback_memo(...)` |
